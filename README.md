# 🎂 Age Calculator

A simple PHP-based web application that calculates your age in years, months, and days. It also wishes you a happy birthday if today is your special day!

## 🚀 Features

- User-friendly interface
- Accurate age calculation
- Birthday greetings
- Responsive design

## 🛠️ Technologies Used

- PHP
- HTML5
- CSS3
